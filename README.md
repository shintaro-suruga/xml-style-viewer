# **xml-style-viewer**
厚生年金の決定通知書等のXMLファイルを閲覧するツール

## これは何をするツールですか？

XML ファイルと、それに対応する XSL ファイルを読み込み、  
**帳票イメージを画面で確認・検索・保存（HTML / PDF）**できるツールです。
厚生年金の決定通知書（公文書）が、Edge標準で閲覧しにくくなったため開発しました。

- XML + XSL をそのままブラウザ感覚で確認できます
- 表示するだけではファイルは保存されません
- 必要なときだけ HTML / PDF として保存できます

---

## 主な機能

### 📂 XML ファイルの表示
- フォルダを指定すると、配下の XML ファイル一覧が表示されます
- XML をダブルクリックすると、右側に帳票イメージが表示されます

### 🔍 文書内検索
- 画面上部の検索欄から、表示中の文書内を検索できます
- 「次を検索」で次の一致箇所へ移動できます
- 「ハイライト解除」で検索結果を消せます

### 💾 保存機能
- **HTML として保存**
- **PDF として保存**

※ 表示しただけではファイルは作成されません  
※ 保存操作を行ったときのみ、XML と同じフォルダに出力されます

### 📁 フォルダ移動（左上ボタン）
左側サイドバー上部に、以下のボタンがあります。

| ボタン | 機能                                     |
| ------ | ---------------------------------------- |
| ←      | 直前に表示していたフォルダへ戻ります     |
| →      | 戻る操作の後、元のフォルダへ進みます     |
| ↑      | 現在のフォルダの一つ上の階層へ移動します |

※ マウスを乗せると説明が表示されます

---

## 安心して使えるポイント

- 表示用のデータは一時領域を使用します  
  → **XML フォルダを勝手に変更しません**
- 保存は必ずユーザー操作が必要です  
  → **意図しないファイル作成は起きません**
- Shift_JIS / UTF-8 など、既存帳票の文字コードに対応しています

---

## 想定される使い方

- 帳票XMLの内容確認
- PDF化して関係者に共有
- 印刷前のレイアウト確認

---

## 注意事項

- XML と XSL は **同じフォルダ**に置いてください
- XML と XSL のファイル名は **同名**である必要があります  
  （例：`henrei.xml` と `henrei.xsl`）

---

## 困ったときは

- XML / XSL の組み合わせが正しいか確認してください
- 表示できない場合は、エラーメッセージの内容を確認してください
